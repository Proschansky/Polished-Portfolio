For this project, I had to use the GIPHY API in order to make buttons which grabbed 10 static, non-animated gif images from the GIPHY API and place them on the page. When the user clicked one of the still GIPHY images, the gif was to animate. If the user clicked the gif again, it was to stop playing. I then was to add a form to my page to the value from a user input and make a button for it appear on the page.
